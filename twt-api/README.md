# First Time Setup

1. Create a venv

```bash
python -m venv venv
```

2. Activate virtual environment
```bash
source venv/Scripts/activate
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python main.py
```
