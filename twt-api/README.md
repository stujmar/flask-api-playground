# Flask API

## In this project I'll be follow this tutorial:
> https://www.youtube.com/watch?v=GMppyAPbLYk

## Create a venv

```bash
python -m venv venv
```

## Activating virtual environment
```bash
source venv/Scripts/activate
```

## Install dependencies

```bash
pip install -r requirements.txt
```

## Run the application

```bash
python app.py
```
