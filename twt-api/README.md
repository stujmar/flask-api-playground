# First Time Setup

1. Create a venv

```bash
python -m venv venv
```

2. Activating virtual environment
```bash
source venv/Scripts/activate
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

## Run the application

```bash
python app.py
```
